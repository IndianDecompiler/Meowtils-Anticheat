# Client-Side Anticheat Mod

A simple Minecraft mod to detect and flag other cheaters.

## Description
This mod is a decompiled and remade version of the anticheat module from the Meowtils project.

## How to use
1. Download the mod file (.jar).
2. Drop the file into your Minecraft mods folder.
3. Launch the game. The anticheat is enabled by default.

## Commands
* /anticheat true - Enable the anticheat.
* /anticheat false - Disable the anticheat.
